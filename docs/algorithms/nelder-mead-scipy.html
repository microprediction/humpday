Nelder-Mead (SciPy)

Simplex Method for Unconstrained Optimization

Authors: Nelder & Mead
Year: 1965
Type: Geometric derivative-free
Best for: General-purpose optimization

Overview

The Nelder-Mead method, also known as the simplex method, is one of the most widely used derivative-free optimization algorithms. It maintains a simplex (a geometric shape with n+1 vertices in n dimensions) and iteratively improves it through reflection, expansion, contraction, and shrinkage operations.

Algorithm Operations

The method uses four fundamental operations to transform the simplex:

Reflection: Mirror the worst point through the centroid
xr = x̄ + α(x̄ - xn+1)
Expansion: Extend further if reflection is successful
xe = x̄ + γ(xr - x̄)
Contraction: Pull back toward centroid if needed
xc = x̄ + β(xworst - x̄)
Shrinkage: Reduce entire simplex toward best point
xi = x1 + σ(xi - x1)

Algorithm Characteristics

Strengths:

  • Simple to implement and understand
  • Robust across many problem types
  • No gradient information required
  • Handles discontinuities reasonably well
  • Good general-purpose performance

Limitations:

  • No convergence guarantees
  • Can be slow near optimum
  • Sensitive to initial simplex construction
  • May struggle with high-dimensional problems
  • Can stagnate on ridges or valleys

Usage Example

from humpday import minimize
import numpy as np

# Optimize Rosenbrock function
def rosenbrock(x):
    return sum(100.0*(x[1:] - x[:-1]**2)**2 + (1 - x[:-1])**2)

# Use Nelder-Mead via HumpDay interface
result = minimize(rosenbrock,
                 x0=np.array([-1.2, 1.0]),
                 method='NelderMead',
                 options={'maxfev': 200})

print(f"Optimum: {result.x}")
print(f"Value: {result.fun}")
print(f"Iterations: {result.nfev}")
