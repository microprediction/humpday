Particle Swarm Optimization (PSO) is a population-based metaheuristic inspired by the social behavior of bird flocking and fish schooling. Developed by Kennedy and Eberhart in 1995, PSO optimizes by having particles move through the search space, with their positions updated based on their own best-known position and the swarm's best-known position.

🔬 Algorithm Theory

  • Velocity Update: v(t+1) = w*v(t) + c1*r1*(pbest - x) + c2*r2*(gbest - x)
  • Position Update: x(t+1) = x(t) + v(t+1)
  • Inertia Weight (w): Controls exploration vs exploitation
  • Cognitive Component (c1): Attraction to personal best
  • Social Component (c2): Attraction to global best
  • Topology: Communication structure between particles

⚙️ PSO Variants

  • Standard PSO: Original Kennedy-Eberhart formulation
  • PSO with Constriction: Clerc-Kennedy constriction factor
  • Comprehensive Learning PSO: Uses multiple exemplars
  • Adaptive PSO: Dynamic parameter adjustment
  • Multi-Swarm PSO: Multiple interacting swarms
  • Quantum PSO: Quantum-inspired position updates
