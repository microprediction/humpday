HumpDay

A collection of derivative-free optimization algorithms in Python and JavaScript

HumpDay provides a unified interface to 22 derivative-free optimization algorithms implemented in pure Python (and also JavaScript). All algorithms operate on the unit hypercube [0,1]ⁿ with automatic domain transformations for bounded and unbounded problems. Each algorithm is cross-validated against an established reference implementation.

Python Usage

Install via pip with minimal dependencies:

pip install humpday

Basic usage for optimization problems:

from humpday import minimize

def objective(x):
    return (x[0] - 2)**2 + (x[1] - 3)**2

result = minimize(objective, bounds=[(-5, 5), (-5, 5)], method='DifferentialEvolution')
print(f"Solution: {result.x}")  # [2.0, 3.0]

Algorithm Categories

Trust Region Methods

Model-based algorithms using quadratic approximations:

  • PRIMA UOBYQA: Unconstrained optimization by quadratic approximation
  • PRIMA NEWUOA: Derivative-free trust-region method (improved UOBYQA)
  • PRIMA BOBYQA: Bound-constrained derivative-free optimization

Classic Numerical Methods

Workhorse non-derivative-free methods, ported to pure Python:

  • Nelder-Mead: Simplex-based direct search
  • Powell: Conjugate direction method
  • LBFGSB: Limited-memory quasi-Newton with box bounds

Evolutionary & Swarm

Population-based stochastic methods:

  • Differential Evolution: Vector-difference mutation with crossover
  • Particle Swarm: Swarm intelligence with velocity updates
  • CMA Evolution Strategy: Covariance matrix adaptation
  • Evolution Strategy: (μ/λ) population search with self-adaptation
  • Genetic Algorithm: Selection, crossover, and mutation operators
  • Firefly Algorithm: Bioluminescence-attraction swarm method
  • Ant Colony Optimization: Pheromone-based pathfinding

Metaheuristics

Acceptance-rule and memory-based search:

  • Simulated Annealing: Probabilistic acceptance with cooling schedule
  • Tabu Search: Memory-based local search with prohibition
  • Harmony Search: Music-inspired pitch-adjustment optimization
  • Bayesian Optimization: Gaussian-process surrogate with expected improvement

Local & Pattern Search

Lightweight baselines and direct-search workhorses:

  • Hill Climbing: Greedy steepest-ascent local search
  • Random Search: Uniform i.i.d. sampling baseline
  • Adaptive Random Search: (1+1)-ES with adaptive step size
  • Coordinate Descent: Axis-aligned line search
  • Pattern Search: Mesh-based direct search without derivatives
