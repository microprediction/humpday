HumpDay

A collection of derivative-free optimization algorithms in Python and JavaScript

Abstract: Humpday provides a unified interface to 22 derivative-free optimization algorithms implemented in pure Python and JavaScript. All algorithms operate on the unit hypercube [0,1]ⁿ with automatic domain transformations for bounded and unbounded problems. Cross-validation against reference implementations ensures mathematical correctness.

Installation and Usage

Install via pip with minimal dependencies:

pip install humpday

Basic usage for optimization problems:

from humpday import minimize

def objective(x):
    return (x[0] - 2)**2 + (x[1] - 3)**2

result = minimize(objective, bounds=[(-5, 5), (-5, 5)], method='DifferentialEvolution')
print(f"Solution: {result.x}")  # [2.0, 3.0]

Algorithm Categories

Trust Region Methods

Model-based algorithms using quadratic approximations:

  • PRIMA UOBYQA: Unconstrained optimization by quadratic approximation
  • PRIMA NEWUOA: Powell's method without derivatives
  • PRIMA BOBYQA: Bound constrained optimization
  • Nelder-Mead: Simplex-based direct search
  • Powell: Conjugate direction method

Evolutionary Algorithms

Population-based stochastic optimization methods:

  • Differential Evolution: Vector-based evolutionary strategy
  • Particle Swarm: Swarm intelligence optimization
  • CMA Evolution Strategy: Covariance matrix adaptation
  • Genetic Algorithm: Selection, crossover, and mutation operators

Metaheuristic Methods

Nature-inspired and local search algorithms:

  • Simulated Annealing: Probabilistic acceptance with cooling schedule
  • Tabu Search: Memory-based local search with prohibition
  • Harmony Search: Music-inspired optimization
  • Firefly Algorithm: Bioluminescence-based swarm method
  • Ant Colony: Pheromone-based pathfinding optimization
