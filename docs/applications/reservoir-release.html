🏞️ Reservoir Release

Over twelve months a reservoir receives seasonal inflow and we choose the monthly release. Releasing generates hydropower, but power per unit release rises with the head, so the same release is worth more when the reservoir is full. Downstream demand must be met and storage must stay within bounds. The optimum holds water to keep the head high while still covering demand. Score is negative hydropower plus penalties, to minimise. 12-D.
